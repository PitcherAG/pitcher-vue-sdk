# Instance Actions
Here you can find actions that you can execute

## Exiting application
```javascript
import { exitApp } from '@pitcher/vue-sdk'

exitApp()
```

## Showing an alert box
```javascript
import { showAlertBox } from '@pitcher/vue-sdk'

showAlertBox('my title', 'my message')
```
